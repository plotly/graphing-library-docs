--- name: Shapes Spanning Subplots language: plotly_js suite: shape order: 12 sitemap: false arrangement: horizontal markdown_content: | *New in 3.4* You can create shapes that span multiple subplots by passing an array of axis references to `xref` and `yref`. Each element in the array specifies which axis the corresponding coordinate refers to. For example, with `xref: ['x', 'x2']`, `x0` refers to the `x` axis and `x1` refers to the `x2` axis. **Note:** When using arrays with `xref` and `yref`, `xsizemode: 'pixel'` and `ysizemode: 'pixel'` are not supported. --- var trace1 = { x: [1, 2, 3], y: [4, 5, 6], mode: 'markers', marker: {size: 10}, xaxis: 'x', yaxis: 'y' }; var trace2 = { x: [1, 2, 3], y: [6, 5, 4], mode: 'markers', marker: {size: 10}, xaxis: 'x2', yaxis: 'y2' }; var layout = { grid: {rows: 1, columns: 2, pattern: 'independent'}, shapes: [ { type: 'rect', // x0 uses x-axis (subplot 1), x1 uses x2-axis (subplot 2) xref: ['x', 'x2'], // y0 uses y-axis (subplot 1), y1 uses y2-axis (subplot 2) yref: ['y', 'y2'], x0: 2, y0: 4.5, x1: 3, y1: 5.5, fillcolor: 'rgba(255, 0, 0, 0.2)', line: {color: 'red', width: 2} } ], showlegend: false }; Plotly.newPlot('myDiv', [trace1, trace2], layout);
