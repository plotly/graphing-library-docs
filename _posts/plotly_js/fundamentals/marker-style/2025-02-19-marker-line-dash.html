--- name: Dashed Marker Borders language: plotly_js suite: marker-style order: 2 sitemap: false arrangement: horizontal markdown_content: | *New in 3.4* Set `dash` on `marker.line` to control the dash pattern of marker borders. Supported values are: `'solid'` (default), `'dot'`, `'dash'`, `'longdash'`, `'dashdot'`, `'longdashdot'`, or a custom dash length list in px (for example, `'12px,6px'`). The `marker.line.dash` attribute is available on `scatter`, `scatterpolar`, `scattergeo`, `scatterternary`, `scattercarpet`, and `scattersmith` traces. --- var data = [{ x: [1, 2, 3, 4, 5], y: [2, 4, 3, 5, 4], type: 'scatter', mode: 'markers', marker: { size: 25, color: 'white', line: { color: 'blue', width: 2, dash: 'dot' } } }]; Plotly.newPlot('myDiv', data)
